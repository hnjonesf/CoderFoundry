CarFinder.

CarFinder is a tool for selecting a vehicle by year, make, model, and trim. It provides photos and reviews.


Backend: WebAPI and MVC
CarFinder is an ASP.NET web application from Visual Studio that created a Single Page Application (SPA). The back end requires secure login for access to the API and resources.
Frontend: AngularJS
To extend HTML5, AngularJS 1.3 is used for a responsive and modular user experience.
Edmunds.com
Edmunds.com is selected for the current api, with access to vehicle information from 1990. Reviews include a summary rating, and the first five user comments.
